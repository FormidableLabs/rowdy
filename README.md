roWDy
=====

A small, rambuctious configuration wrapper for
[WD.js](https://github.com/admc/wd).

## Development

### Installation

Install with [npm](https://www.npmjs.org/):

    $ npm install

### Build

We follow a [Gulp](http://gulpjs.com/) workflow. To check everything:

    $ gulp check
